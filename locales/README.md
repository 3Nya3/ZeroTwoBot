## Creating a locale
---
Keep in mind the following:
* Locale id must be 2 characters long (lowercase)
* Placeholders can be specified like C# ex: `"My name is {0} nice to meet you."`
* Emotes can be stated in their plaintext: `1 + 1 = 3 :upside_down:`
